# OpenGP Keyboard Shortcuts

> Press `?` at any time to see shortcuts for your current screen.

This guide covers all keyboard shortcuts in OpenGP. Learn these to work faster and navigate the application with ease.

---

## Quick Reference

| Action | Key |
|--------|-----|
| Show shortcuts for current screen | `?` |
| Quit application | `q` or `Ctrl+C` |
| Create new (patient/appointment) | `n` |
| Search | `/` |
| Navigate up/down | `j` / `k` or `‚Üë` / `‚Üì` |
| Go to first item | `g` |
| Go to last item | `G` |
| Cancel / Close | `Esc` |
| Confirm / Submit | `Enter` |
| Next field | `Tab` |
| Previous field | `Shift+Tab` |

## Global Shortcuts

These shortcuts work from any screen in the application:

| Key | Action |
|-----|--------|

## Patient Management

### Patient List Screen

When viewing the list of patients:

| Key | Action |
|-----|--------|
| `j` or `‚Üì` | Select next patient |
| `k` or `‚Üë` | Select previous patient |
| `g` | Jump to first patient |
| `G` | Jump to last patient |
| `n` | Create new patient |
| `/` | Start searching |
| `Esc` | Clear search or exit search mode |
| `Enter` | View patient details |

#### Searching Patients

When the search box is active:

| Key | Action |
|-----|--------|
| Type any letter | Add to search query |
| `Backspace` | Remove last character |
| `‚Üë` / `‚Üì` | Navigate through results |
| `Enter` | Exit search mode |
| `Esc` | Exit search mode |

### Patient Form

When creating or editing a patient:

| Key | Action |
|-----|--------|
| `Tab` | Move to next field |
| `Shift+Tab` | Move to previous field |
| `‚Üë` / `‚Üì` | Navigate between fields (or cycle through options like gender) |
| `Enter` | Save and close form |
| `F10` | Save and close form |
| `Esc` | Cancel and close form |

**Tip**: Your changes are saved when you press Enter or F10. Press Esc to cancel without saving.

### Appointment Calendar

The calendar has two views: **Month View** (overview) and **Day View** (detailed schedule).

#### Month View

Navigate the monthly calendar:

| Key | Action |
|-----|--------|
| `‚Üë` | Go to previous week |
| `‚Üì` | Go to next week |
| `‚Üê` | Go to previous day |
| `‚Üí` | Go to next day |
| `h` | Go to previous month |
| `l` | Go to next month |
| `t` | Jump to today |
| `n` | Create new appointment |
| `Enter` | Switch to day view |

#### Day View

When viewing a specific day's schedule:

| Key | Action |
|-----|--------|
| `j` or `‚Üì` | Move to next time slot |
| `k` or `‚Üë` | Move to previous time slot |

| `Esc` | Switch to month view |

| `c` | Mark appointment as Completed |

**Important**: When you have an appointment selected in day view, pressing `n` marks it as "No Show" instead of creating a new appointment. To create a new appointment, press `n` in month view first.

## Tips & Tricks

### Navigation Patterns

- **Lists**: Use `j`/`k` or arrow keys to move up and down. It's faster than using arrows!
- **Quick Jump**: Press `g` to jump to the first item, `G` for the last.
- **Today**: Press `t` from the calendar to instantly jump to today's date.

### Working with Forms

- **Tab Navigation**: Use `Tab` to move forward, `Shift+Tab` to move backward.
- **Quick Save**: Press `Enter` in forms to save and close. `F10` works too.
- **Cancel**: Press `Esc` to cancel or close anything.

### Calendar Productivity

- **Switch Views**: Press `v` to toggle between day and week views.
- **Status Updates**: Quickly mark appointments with `a` (Arrived), `c` (Completed), or `n` (No Show).
- **Batch Operations**: Use `m` to enter multi-select mode, then press `b` for batch operations.
- **Undo Mistakes**: Made a status change by accident? Press `Ctrl+Z` within 30 seconds to undo.

### Finding Things

- **Search**: Press `/` to search from most screens.
- **Filters**: Use `f` to filter appointments by status, `p` to filter by practitioner.
- **Help**: Always press `?` to see what's available on your current screen!
